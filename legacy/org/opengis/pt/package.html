OpenGIS® Positioning specification. This package contains classes and interfaces specified in the OpenGIS® Coordinate Transformation Services Implementation Specification.

THE CONTENT OF THIS PACKAGE IS DEPRECATED FOR NOW. THIS PACKAGE WAS BUILD ON TOP OF VERSION 1.0 OF OpenGIS® SPECIFICATIONS, WHICH IS INCOMPATIBLE WITH LATEST VERSION (2.0). The specification changed in an incompatible way because (quote from OpenGIS® document):

The reasons for replacement of OGC Document 99-102r1 were:

  1. The publication of ISO 19111, which covers largely the same ground as OGC Document 99-102r1 and the alignment with that document.

  2. Alignment with published Implementation Specification "OpenGIS® Coordinate Transformation Services Implementation Specification" and OpenGIS® Recommendation Paper: "Recommended Definition Data for Coordinate Reference Systems and Coordinate Transformations". The abstract model that underlies both of these documents has progressed considerably from that described in OGC Document 99-102r1 and is more in line with ISO 19111.
