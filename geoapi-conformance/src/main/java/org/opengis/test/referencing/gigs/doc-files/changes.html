Changes compared to GIGS files

The changes applied to GIGS test files fall in three categories:

  • Replacement of deprecated definitions: if the EPSG dataset specifies a replacement for a deprecated object, the definition is modified to match the new object.
  • Renaming: if an object name changed in newer releases of EPSG dataset, the name in the GIGS file is updated to match the new name.
  • Accuracy: GIGS guidance notice typically requires an accuracy of 10 decimal figures. However some data declared in the original Excel files do not allow such precision. This page documents the necessary changes applied in order to meet the required accuracy. All new values listed below come from the EPSG dataset version 7.9 or later.
