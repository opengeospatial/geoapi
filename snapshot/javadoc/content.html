GeoAPI content

This page lists all non-deprecated GeoAPI 3.1-SNAPSHOT methods and fields, together with the identifiers specified in the international standards published by OGC and ISO. Versions of the standard used by GeoAPI are documented in the Specification javadoc. Version different than the default one are specified by a colon followed by the publication year, for example “ISO 19115:2003”.

Differences between the OGC/ISO identifiers and the GeoAPI name are mostly for compliance with Java usage, for example by adding the get prefix. More significant name changes are emphased by italics.

The “Note” column can contain the following values:

(N) for new methods added in GeoAPI 3.1-SNAPSHOT (sometime merely renaming of older methods).
(I) for incompatible changes compared to the previous GeoAPI release (should be very rare in minor releases).
(MC) for methods that may change in an incompatible way in the next major release. Those changes are not applied in this release, but are flagged for allowing developers to anticipate.
